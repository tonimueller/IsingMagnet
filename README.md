# README: Uebung 1 - Ising Modell

# Compilation
```shell 
mkdir build 
cd build 
cmake .. 
make 
```
# Execution 
Go to the Analyze directory and use the shell and jupyter-notebook to start the simulation and plot the data. 
```shell 
cd Analyze 
bash executeCode.sh
```
Afterwards open the notebook for the analyze.
